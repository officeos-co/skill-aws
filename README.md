# AWS

Manage AWS cloud infrastructure: EC2, S3, Lambda, IAM, CloudWatch, ECS, Route53, SQS, SNS, and RDS via the AWS REST API.

## Categories

- Cloud Infrastructure

## Credentials

| Key | Label | Kind |
|-----|-------|------|
| `access_key_id` | Access Key ID | text |
| `secret_access_key` | Secret Access Key | password |
| `region` | Default Region | text |

This skill is included in [EnterpriseAgentOS](https://officeos.co) and requires no separate installation.
